
# React TODO App (Pure CSS)

Bu proje React kullanılarak geliştirilmiş basit bir TODO uygulamasıdır.

## Özellikler
- Görev Ekleme
- Görev Listeleme
- Görev Güncelleme
- Görev Silme

## Kullanılan Teknolojiler
- ReactJS
- Pure CSS

## Kurulum

```
npm install
npm start
```

## Netlify Yayınlama

1. Github'a yükle
2. Netlify -> Import from Github
3. Deploy
